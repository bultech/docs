# Anticipated Cost Report Settings

This section allows you to manage the general behavior, scenarios, and templates of the **[Anticipated Cost Report](/en/cost-management/anticipated-cost-report)**.

## Accessing the Settings Panel

You can access the settings panel by clicking the **Settings** (<img src="/icons/cog-outline.svg" alt="cog-outline" className="inline align-middle dark:invert" style={{width: '16px', height: '16px', margin: 0}} />) icon in the top right corner of the **Anticipated Cost Report** page.

## Settings and Scenario Management

<AccordionGroup>
  <Accordion title="Enable Scenario-Based Reporting">
    - **Function:** By checking this option, you can create and manage multiple scenarios (e.g., "Optimistic Finish," "Pessimistic Finish") instead of a single default report.
    - **Effect:** When this feature is activated, a **Scenario Management** (<img src="/icons/file-tree-outline.svg" alt="file-tree-outline" className="inline align-middle dark:invert" style={{width: '16px', height: '16px', margin: 0}} />) button is added to the top right corner of the report page.
  </Accordion>
  <Accordion title="Scenario Management">
    When scenario-based reporting is active, you can manage your scenarios by clicking the **Scenario Management** (<img src="/icons/file-tree-outline.svg" alt="file-tree-outline" className="inline align-middle dark:invert" style={{width: '16px', height: '16px', margin: 0}} />) button in the top right.
    - **Interface:** The opened window contains a scenario list, a button to create a new scenario, and an apply button.
    - **Actions:**
        - **Select Scenario:** You can select a scenario by clicking on it in the list.
        - **Edit/Delete Scenario:** You can manage a scenario with the **Edit** (<img src="/icons/pencil-outline.svg" alt="pencil-outline" className="inline align-middle dark:invert" style={{width: '16px', height: '16px', margin: 0}} />) and **Delete** (<img src="/icons/trash-can-outline.svg" alt="trash-can-outline" className="inline align-middle dark:invert" style={{width: '16px', height: '16px', margin: 0}} />) icons that appear when you hover over it.
  </Accordion>
</AccordionGroup>
